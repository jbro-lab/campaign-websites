# Image Placeholders — Single Page Hero Template

Place your campaign images in this folder. Below are the recommended sizes for each image slot.

## Required Images

| Image | Recommended Size | Format | Notes |
|-------|-----------------|--------|-------|
| **Hero candidate photo** | 600 x 750 px | JPG or PNG | Portrait orientation. High-quality headshot or professional campaign photo. Name it `candidate-hero.jpg` |
| **About section photo** | 500 x 600 px | JPG or PNG | Portrait orientation. Casual/community photo (e.g., at an event, with constituents). Name it `candidate-about.jpg` |

## Tips

- Use high-resolution images (at least 2x the listed dimensions for retina displays)
- JPG is best for photos (smaller file size). PNG if you need transparency.
- Keep file sizes under 500KB each for fast loading — use a tool like TinyPNG to compress
- Make sure filenames match what you put in the HTML `<img>` tags (capitalization matters!)
